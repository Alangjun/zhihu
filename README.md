# 仿知乎官网
## 开发工具
HTML5，CSS3，JavaScript，JQuery，Bootstrasp，Ajax，PHP，Mysql，Photoshop
## 描述
基本模仿知乎官网界面，现已具备登录、注册，提问，评论，页面导航栏及侧栏固定等功能，其中，登录及注册都需要验证，提问及评论的内容选择性地显示在界面中，其他功能正逐步完善中
